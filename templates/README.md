# Problem Templates

This directory contains Python template files for various system design problems.

## Files

- `python_designproblems_all.py` - Complete collection of design problems in Python
- `python_lrucache_complete.py` - LRU Cache implementation template
- `python_ratelimiter_all.py` - Rate Limiter implementation template

## Usage

These templates serve as reference implementations and can be used to:
- Generate problem content
- Provide solution examples
- Test algorithm implementations
